# Product Roadmap

## MVP (0–3 months)
- [ ] Core editor with canvas (Fabric.js) + basic elements
- [ ] Hand-draw SVG path animation (pencil, marker)
- [ ] Scene management + timeline (drag/drop)
- [ ] Asset library (built-in SVG + icon set)
- [ ] Audio upload + voiceover record
- [ ] MP4 export via FFmpeg (1080p)
- [ ] Email/password auth + Google OAuth
- [ ] Project dashboard (CRUD, thumbnails)
- [ ] 3 whiteboard templates
- [ ] Stripe billing (Free / Starter / Pro)
- [ ] Docker deployment

## Growth (3–6 months)
- [ ] AI script generator (GPT-4o)
- [ ] AI TTS voiceover (OpenAI + ElevenLabs)
- [ ] Extended character library (500+ SVG characters)
- [ ] Template marketplace (50+ templates)
- [ ] Real-time collaboration (Yjs CRDT + WebSocket)
- [ ] Comments + version history
- [ ] WebM / GIF export
- [ ] 4K export tier
- [ ] Team workspaces + invite system
- [ ] Admin dashboard
- [ ] Kubernetes deployment + CI/CD

## Scale (6–12 months)
- [ ] Oil & Gas training module (full asset library)
- [ ] AI image generation (DALL-E 3 / Stable Diffusion)
- [ ] AI scene suggestions per script segment
- [ ] Microsoft OAuth + SSO/SAML for enterprise
- [ ] Custom brand kit (fonts, colors, logos)
- [ ] White-label mode
- [ ] API access for enterprise customers
- [ ] Offline export worker fleet (auto-scaling)
- [ ] Multi-region deployment (US, EU, Middle East)
- [ ] GDPR + SOC 2 Type II compliance
- [ ] Analytics dashboard for video performance

## Enterprise (12+ months)
- [ ] On-premise deployment option
- [ ] Custom AI model fine-tuning (per-industry)
- [ ] LMS integration (SCORM, xAPI)
- [ ] Advanced O&G module: interactive simulations
- [ ] Certification tracking integration
- [ ] Enterprise SLA + dedicated support
- [ ] Custom animation style libraries
- [ ] Morphing animation engine (SVG morphing)
